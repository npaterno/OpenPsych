This package contains data collected during the 2026 SIPS @ SPSP Hack-a-thon for Open Data. Data will be migrated into the [OpenIntro](https://github.com/OpenIntroStat/openintro) package.

## Installation

You can install the development version of OpenPsych from
GitHub with:

``` r
#install.packages("devtools")
library(devtools)
install_github("npaterno/OpenPsych")
```
